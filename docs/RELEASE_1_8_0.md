# Vaultarr 1.8.0 — Transforming Museum Scan Pill

Vaultarr's permanent Museum scan control now has two distinct visual identities without changing its dimensions or shifting the sidebar.

## Changes

- Increased vertical spacing between **Scan Museum** and **Search Museum**.
- Preserved the compact fixed-size pill used throughout the application.
- The idle state remains a clear blue primary action.
- The working state becomes a darker, quieter status surface with live stage, current game, percentage, and progress.
- Added a restrained animated sheen and slow scan icon movement while work is active.
- Added separate green completion and muted failure states.
- Completion still returns smoothly to the regular **Scan Museum** action.
- Page load and navigation remain status-only and cannot start a scan.

## Protected systems

No changes were made to floating-card hover or tilt, expanded cards, Focus Mode, sidebar navigation motion, Universal Search, manuals, Acquisition Assistant, or the Museum scan backend.
