# Vaultarr 1.7.1 — Primary Sidebar Scan Control

The permanent **Scan Museum** control now sits directly above **Search Museum** and uses the same strong blue primary-action language as the Home page's **Enter Museum** button.

## Changes

- Moved Scan Museum into the sidebar utility stack immediately above Search Museum.
- Restyled the idle scan control as a full-width primary blue gradient button.
- Preserved the smooth in-place expansion, progress details, completion state, and contraction behavior.
- Kept the active scan state calmer and readable while retaining the Vaultarr blue aesthetic.
- No changes to scanning logic or protected card, expansion, Focus Mode, search, manual, or navigation systems.
