# Installation

## Windows local run

1. Install Python.
2. Extract Vaultarr.
3. Install requirements:

```bash
pip install -r requirements.txt
```

4. Start Vaultarr:

```bash
python -m app.main
```

## Docker

Use one of the compose templates in `docker/` or the root `docker-compose.yml`.

## First run

Vaultarr opens the setup wizard when no vault has been configured yet.
