# Vaultarr 2.0.0 Alpha 20 — Full-Card Cover Stretch

This release replaces the adaptive Alpha 19 cover presentation with a uniform full-card treatment.

## Changes

- Removed the blurred background duplicate.
- Removed `object-fit: contain`.
- Cover artwork now uses `object-fit: fill`.
- Images stretch or compress to match the card dimensions exactly.
- No artwork is cropped and no letterboxing remains.

## Protected systems

No changes were made to floating-card hover, 3D tilt, parallax, gloss, expansion, Focus Mode, Museum Scan, manuals, Acquisition Assistant, or sidebar navigation.
