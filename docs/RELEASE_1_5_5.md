# Vaultarr 1.5.5 — Performance Foundation

This release reduces frontend work on lower-end systems without changing Vaultarr's visual identity. It adds lazy cover loading, offscreen rendering containment, a shared frame-throttled card hover controller, debounced library search, hidden-tab activity suspension, and conditional Focus Mode script loading.

No cataloging, metadata, manuals, acquisition, or navigation behavior was intentionally changed.
