# Vaultarr 2.0.0 Alpha 52 — Milestone Journey

Alpha 52 transforms Milestones into a calm, legible museum-progress dashboard that makes current status and the next useful action immediately clear.

## Changes

- Leads with the museum's earned badges, catalogued games, preserved games, overall milestone progress, and closest milestone.
- Places the four closest goals directly beneath the hero with clear percentages, counts, descriptions, and progress bars.
- Presents the full achievement archive with distinct unlocked and in-progress states.
- Expands the archive from preservation-only signals to emulator launches, owned soundtrack tracks, saved trailers, gallery coverage, collections, authorized local acquisitions, and integrations.
- Records local Vaultarr player launches so emulator achievements reflect real use.
- Gives each achievement family its own color identity and adds subtle staggered symbol motion with a reduced-motion fallback.
- Derives percentage-liquid colors from the active theme for Milestone Progress, Discovery Depth, Home museum health, and Museum Scan.
- Replaces the older collection, publisher, decade, and sound-preview panels with one coherent Curator Paths section.
- Summarizes live achievement progress across cataloging, preservation, visual exhibits, soundtrack archiving, in-app play, and connected metadata.
- Reorganizes Discover below Release Timeline into a Curator Index, Quiet Corners, and Guided Shelves journey.
- Gives developer, publisher, genre, rediscovery, museum-care, and install-size browsing one calm responsive visual system.
- Reframes Museum views, search, filters, sorting, and result context as one unified archive-control chamber.
- Preserves Museum cards, card interaction physics, search and filter behavior, routes, and Add Game behavior.
- Adds responsive layouts for desktop, tablet, and narrow screens.

## Preserved systems

- The protected outer housing and inner glass Milestone vessel are unchanged.
- Canvas liquid physics, percentage-based fill level, bubble-driven motion, and clipping are unchanged.
- Theme switching changes liquid paint colors only; spring behavior and percentage data remain unchanged.
- The curved Milestone Progress label and milestone percentage calculation are unchanged.
- Existing badge targets, routes, sidebar, Museum cards, and unrelated systems are unchanged; additional achievement families now contribute through the same protected percentage formula.
