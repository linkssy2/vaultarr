# Vaultarr 1.7.4 — Refined Pill Scan Control

This release makes the two sidebar utilities visually consistent with the rest of Vaultarr.

## Changes

- **Scan Museum** and **Search Museum** now use matching compact pill dimensions.
- The scan pill expands vertically in place without changing sidebar width.
- Stage, percentage, progress, and current-game information fade in after expansion begins.
- Completion details fade first, followed by a smooth contraction back to the idle pill.
- Mobile sizing was tightened to preserve the same visual hierarchy.

## Protected systems

No changes were made to floating-card tilt, card expansion, Focus Mode, sidebar navigation motion, Universal Search, manuals, Acquisition Assistant, or the scan engine itself.
