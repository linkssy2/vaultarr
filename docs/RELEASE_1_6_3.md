# Vaultarr 1.6.3 — Background Museum Scan

Vaultarr now uses a single Plex-style Museum Scan instead of exposing individual preparation jobs in Standard Mode.

## Highlights
- Scan Museum button on the Museum page and Home lobby.
- One persistent global progress indicator that follows navigation.
- Folder scanning, incomplete-record detection, and preparation run in the background.
- Completion summary updates without a page reload.
- Activity removed from the standard sidebar and Home page.
- Legacy `/activity` and `/curator` routes redirect to Museum.
- Individual curator jobs remain internal for compatibility and advanced integrations.
- Protected 1.5.8 card, tilt, expansion, Focus Mode, and navigation systems are unchanged.
