# Vaultarr UX Baseline

**Protected baseline:** Vaultarr 1.5.8  
**Museum experience release:** Vaultarr 1.6.0

Future releases should extend around these systems instead of rewriting them.

## Floating Card Engine
Preserve hover lift, pointer-driven 3D tilt, cover parallax, gloss, dynamic shadow, selection, and expansion handoff.

## Expanded Card / Focus Mode
Preserve smooth open and close motion, tab behavior, Acquisition Assistant, manuals, and the absence of full-page refreshes.

## Sidebar Navigation
Preserve preparation-first navigation, established timing and easing, cache behavior, and stale-page refresh logic.

## Universal Search
Preserve immediate focus, My Museum / Discover switching, usable controls after reopening, and live Add to Museum behavior.

## Curator Progress
Preserve immediate feedback, smoothly interpolated progress, background-job polling, row-only updates, and no reload at completion.

## Manual System
Preserve cached VideoGameManual search, live Vimm manual search, cross-platform matching, adaptive DPI, and PDF validation.

## Live Museum State
Preserve add, update, and removal synchronization without requiring a browser refresh.

## Release check
1. Card hover, tilt, parallax, expand, and close.
2. Repeated sidebar navigation.
3. Universal Search switching and reopening.
4. Activity progress from start through completion.
5. Manual search and download.
6. Add to Museum while on and away from the Museum page.
