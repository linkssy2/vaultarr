# Vaultarr 2.0.0 Alpha 48 — Museum Home Actions

Alpha 48 brings the three lower Home destinations into Vaultarr's museum interface language.

## Refined actions

- Continue curating uses a collection-review glyph and retains the live open-task count.
- Timeline uses a restrained timepiece glyph and continues to open the Discovery timeline.
- Next milestone uses a milestone glyph and retains the live museum game count.
- Each action uses a glass chamber, slim illuminated edge, subtle orbital detail, and its own restrained accent color.
- Titles, supporting details, and directional controls now form a clearer and more consistent hierarchy.
- Universal Search no longer displays the redundant empty-state Discover & Add instruction; live search progress, results, and errors remain available.

## Interaction and protection

- Hover and keyboard focus gently lift the chamber, illuminate its icon, and move the direction control.
- The layout remains responsive and stacks cleanly on narrower screens.
- The Home Orbital, hero, recent collection shelf, routes, calculations, sidebar, and unrelated pages remain unchanged.

## Validation target

- Confirm all three actions remain equal in size and align across the Home page.
- Confirm each action preserves its original route and live value.
- Confirm focus, hover, and responsive states remain calm and readable.
- Run Python compilation, template compilation, JavaScript syntax checks, and browser inspection.
