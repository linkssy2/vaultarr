# Vaultarr 1.6.5 — Living Sidebar

Vaultarr 1.6.5 refines the Museum Scan indicator so it feels like a natural part of the application rather than a notification that abruptly appears and disappears.

## Changes

- Smooth sidebar scan entrance with a soft slide, expansion, and fade.
- Smooth completion sequence that compresses, holds briefly, and exits cleanly.
- Frame-by-frame progress and percentage interpolation.
- Crossfading stage and game-detail text without layout jitter.
- Subtle breathing scan indicator and completion confirmation animation.
- Refined hover treatment using the existing Vaultarr blue glow and restrained lift.
- Click-to-expand scan details retained.
- Reduced-motion compatibility retained.

The scan engine and the protected 1.5.8 card, tilt, expansion, Focus Mode, navigation, Universal Search, manuals, and Acquisition Assistant systems are unchanged.
