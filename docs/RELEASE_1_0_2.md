# Vaultarr 1.0.3 — Reset Vault

## Added
- Reset Vault danger-zone action in Studio.
- Typed confirmation requirement: `RESET VAULT`.
- Optional cached media/manual/provider-index removal.
- Optional Time Capsule removal, off by default.
- Reset returns Vaultarr to first-run onboarding.

## Fixed
- Fresh installs and reset vaults now redirect to onboarding from Home when no library roots exist.
