# Vaultarr 2.0.0 Alpha 22 — Milestones Simplification

This focused update removes redundant navigation from the Milestones hero.

## Changes

- Removed **Improve Your Museum**.
- Removed **Rediscover**.
- Reduced the hero's vertical footprint so milestone content appears sooner.
- Kept badges, progress, achievements, Vault Legacy score, and audio feedback unchanged.

## Protected systems

No changes were made to Museum cards, hover, tilt, parallax, expansion, Museum Scan, Universal Search, manuals, Acquisition Assistant, the Orbital Core, or sidebar navigation.
