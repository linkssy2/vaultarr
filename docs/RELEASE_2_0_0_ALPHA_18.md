# Vaultarr 2.0.0 Alpha 18 — Settings Danger Zone

This focused release moves **Reset Vault Settings** to the absolute bottom of the Settings page.

## Changes

- Museum Backup remains above the destructive reset controls.
- Reset Vault Settings now appears in a visually isolated Danger Zone.
- Existing confirmation, cached-asset deletion, Museum Backup retention, and reset behavior are unchanged.
- Responsive spacing keeps the section distinct on smaller displays.

## Protected systems

No changes were made to Museum cards, hover/tilt, expansion, Museum Scan, Universal Search, manuals, Acquisition Assistant, authentication, or the Orbital Core.
