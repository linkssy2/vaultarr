# Vaultarr 1.1.9 — Mobile Refinement

This release expands the responsive compatibility work introduced in 1.1.8 and targets the remaining pages and controls that were difficult to use on phones.

## Improvements
- Refined responsive layouts across Home, Library, Collections, Discovery, Milestones, Preservation, Time Capsule, Studio, and health views.
- Improved phone-sized game grids and card spacing.
- Added horizontal scrolling for chip and filter rows instead of wrapping into oversized blocks.
- Improved action rows, forms, headers, panels, and long-path handling.
- Refined Focus Mode, gallery, metadata, preservation, and modal layouts.
- Improved table behavior, search overlays, small-screen typography, and touch targets.
- Reduced Orbital Vault sizing on narrow devices while preserving its animation.

## Stability
No database, scanning, metadata, authentication, preservation, or API behavior was changed. The release is limited to responsive presentation and version metadata.
