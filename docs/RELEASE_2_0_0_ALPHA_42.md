# Vaultarr 2.0.0 Alpha 42 — Calm Search and Simpler Settings

Alpha 42 makes Universal Search feel continuous and reorganizes Settings around the tasks people actually need.

## Universal Search motion

- My Museum and Discover & Add now share one indicator that slides smoothly between modes.
- Discover controls reveal calmly instead of appearing abruptly.
- Information Source and game-category menus use Vaultarr's glass chamber reveal with staggered option motion.
- Dropdown menus may extend beyond the Universal Search surface without being clipped by its rounded enclosure.
- Closing by backdrop, Escape, or the close button now completes the panel and backdrop transition before accessibility state hides the dialog.

## Simplified Settings

- A task hub provides direct shortcuts to Experience, Game Folders, Appearance, Access, Museum Backup, and Advanced tools.
- Everyday Experience and Game Folders remain open by default.
- Secondary sections use accessible Show and Hide controls and open automatically when reached from a shortcut.
- Existing forms, submission routes, values, Advanced Mode visibility, authentication behavior, and Museum Backup behavior are unchanged.

## Validation target

- Verify Universal Search mode switching, both custom dropdowns, and all three closing paths.
- Verify Settings shortcuts, section expansion, forms, Standard Mode visibility, and keyboard focus.
- Run Python compilation, template compilation, and JavaScript syntax checks.
