# Vaultarr 2.0.0 Alpha 49 — Login Orbital Motion

Alpha 49 restores visible motion to the unified login orbital for visitors whose browser or operating system requests reduced motion.

## Changes

- The five login orbital rings continue rotating at gentler, individually paced speeds under reduced-motion preferences.
- The central energy ring continues a slow rotation so the Vaultarr core never appears frozen.
- Comet, nebula, scan-line, breathing, pulsing, and node-glow effects remain disabled in reduced-motion mode.
- Successful login now eases the access chamber into Vaultarr's dark canvas before the Home shell and content settle into view.
- Invalid credentials return through a restrained login fade instead of flashing the page back into place.
- Login asset cache keys now identify Alpha 49 so browser and Docker refreshes load the corrected CSS.

## Preserved behavior

The unified login layout, canonical Vaultarr logo, credential verification, session handling, redirects, logout, authentication settings, and unrelated protected systems are unchanged.
