# Vaultarr 2.0.0 Alpha 21 — Discover Simplification

This focused update removes redundant navigation from the Discover hero.

## Changes

- Removed **Browse Museum**.
- Removed **Improve Museum**.
- Reduced the hero's vertical footprint so Discover content appears sooner.
- Kept the Timeline, shelves, search, filters, cards, and add-to-Museum workflow unchanged.

## Protected systems

No changes were made to Museum cards, floating hover, tilt, parallax, expansion, Museum Scan, manuals, Acquisition Assistant, the Orbital Core, or sidebar navigation.
