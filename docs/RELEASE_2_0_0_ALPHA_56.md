# Vaultarr 2.0.0 Alpha 56

Alpha 56 makes removing a cataloged game feel like a calm museum deaccession instead of a page reload.

## Removal experience

- The expanded exhibit closes smoothly after confirmation.
- The removed card fades out and neighboring cards glide into their new positions.
- Museum result counts and search state update without navigation.
- Removing the final visible game reveals a helpful empty state in the existing Museum shell.
- Attention views clear their summary when their final game is removed.
- A confirmation message reminds the curator that original game files were untouched.

## Preserved behavior

- The existing removal confirmation and options remain unchanged.
- Ignore-path and cached-asset choices still reach the same removal service.
- No routes, Museum card dimensions, hover effects, sidebar behavior, or protected systems were changed.
