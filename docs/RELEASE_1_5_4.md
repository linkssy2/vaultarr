# Vaultarr 1.5.5 — Expanded Card Acquisition Tab

The Acquisition Assistant is now available as a dedicated tab inside every expanded game card. Users can search the live Vimm reference catalog, choose a release, open the exact source page, paste the final link, save it, and attach local files without leaving the Library.

The full game-details page remains available as a secondary path. Existing Curator, manuals, card expansion, Focus Mode, and navigation systems are unchanged.
