# Vaultarr 1.1.12 — Focus Mode Animation Hotfix

Vaultarr 1.1.12 fixes the mobile card-expansion regression introduced in 1.1.11.

## Fixed

- Restored the known-good 1.1.10 Focus Mode expansion engine.
- Restored the smooth mobile open and close transition contract.
- Prevented newer mobile layout rules from overriding Focus Mode geometry during expansion.
- Restored the floating mobile close control and original artwork spacing.
- Preserved the broader 1.1.11 mobile navigation, search, forms, safe-area, touch, and layout improvements.

## Stability

This patch only changes frontend JavaScript, responsive CSS, version metadata, and documentation. It does not modify the database, authentication, scanning, metadata providers, preservation logic, archive operations, or file handling.
