# Vaultarr 1.7.9 — Morphing Museum Scan Pill

The sidebar Museum Scan control no longer opens a drawer or changes size. The existing pill smoothly morphs into a compact live progress display, then returns to its idle Scan Museum state after completion.

## Changes
- Fixed 44 px pill footprint at every stage
- No sidebar reflow or layout jump
- Crossfading idle and live states
- Embedded stage, current game, percentage, and progress line
- Smooth completion and return to idle
- Manual scan start only
