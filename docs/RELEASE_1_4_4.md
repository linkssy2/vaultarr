# Vaultarr 1.4.4 — Universal Search & Add

Vaultarr now uses the existing sidebar search as the single place to find both games already in the museum and games available through enabled information sources.

## Highlights
- Toggle between **My Museum** and **Discover & Add**.
- Search enabled metadata sources without leaving the current page.
- Preview and add a release directly from search results.
- Choose an optional local folder and category before adding.
- Automatically queue imported records for Curator preparation.
- Use Library → Add Game for detailed manual entry when a title cannot be found.

## Docker
Production deployments continue to pull `ghcr.io/linkssy2/vaultarr:latest`. Local `build: .` usage is reserved for development.
