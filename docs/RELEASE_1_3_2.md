# Vaultarr 1.3.2 — UI Component Polish

This release replaces inconsistent browser-native checkboxes with a unified Vaultarr control system.

## Improvements

- Binary settings now use a spacious animated VaultToggle.
- Compact selections use a larger rounded VaultCheckbox.
- Controls share consistent hover, press, focus, enabled, and disabled states.
- Entire setting cards remain clickable for easier desktop and mobile use.
- Provider, security, compatibility, archive, manual, and metadata-lock controls now follow one visual language.
- Keyboard focus and reduced-motion behavior are preserved.

## Stability

No changes were made to Curator processing, metadata matching, manual providers, card expansion, Focus Mode, or sidebar navigation motion.
