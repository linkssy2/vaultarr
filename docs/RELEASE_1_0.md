# Vaultarr 1.0

This release freezes the current feature set and shifts the project into product readiness.

## Added for 1.0

- First-run onboarding page at `/onboarding`
- System Health page at `/health`
- Database optimize action
- Dockerfile and docker-compose template
- README with first-run and Docker setup
- Build label updated to Vaultarr 1.0

## 1.0 Philosophy

Vaultarr treats each game as a digital artifact: metadata, manuals, media, trailers, patches, preservation state, and collection context are all part of the record.
