# Vaultarr 2.0.0 Alpha 2 — Home Action Clarity

This experimental refinement reduces duplicate Home navigation and improves lower-sidebar cohesion.

## Changes

- Removed **Open Museum** from the Home header.
- Removed **View Museum** from the Continue Browsing shelf.
- **Continue Curating** now remains the single action for incomplete exhibits.
- **Discover Games** remains the single action for finding new titles.
- Continue Browsing opens its featured game directly.
- Search Museum stays on one line, truncates safely, and keeps the Ctrl K keycap aligned.
- Added consistent spacing between Scan Museum, Search Museum, and build status.

## Stable systems retained

Floating card hover and tilt, Focus Mode expansion, smooth navigation, manuals, Acquisition Assistant, Curator services, and Museum Scan behavior are unchanged.
