# Vaultarr 2.0.0 Alpha 51 — Complete Platform Menu

Alpha 51 corrects the Museum platform dropdown so the platform list remains readable and its scroll window ends on a complete option.

## Changes

- Widened only the platform list to accommodate full console and computer platform names.
- Kept the compact platform filter trigger at its existing size.
- Aligned the menu's maximum height to five complete option rows before scrolling.
- Added safe ellipsis handling for exceptionally long platform names.
- Preserved the existing glass styling, glide-in option motion, filtering behavior, Museum layout, and protected card interactions.

## Validation

- Python modules compile successfully.
- JavaScript assets pass syntax validation.
- Jinja templates parse successfully.
- The Museum platform menu was inspected in the running application at desktop and narrow viewport widths.
