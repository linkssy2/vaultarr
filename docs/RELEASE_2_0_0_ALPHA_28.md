# Vaultarr 2.0.0 Alpha 28 — Liquid Milestone Vessel

## Visual redesign

- Progress is represented by liquid height inside the circle.
- 9% progress fills the bottom 9% of the vessel.
- 50% reaches the midpoint.
- 100% fills the vessel completely.
- Removed the circular bar, outer orbit line, and energy bead.
- Added bright Vaultarr-blue liquid, a glowing top edge, subtle surface motion, and a breathing halo.

## Preserved

Milestone calculations, badge targets, achievements, Museum cards, scan controls, manuals, Acquisition Assistant, and navigation remain unchanged.
