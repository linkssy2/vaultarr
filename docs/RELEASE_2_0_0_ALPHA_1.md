# Vaultarr 2.0.0 Alpha 1 — Familiar Museum Experience

An experimental presentation branch built from the stable 1.8.4 baseline.

## Goals
- Make the first 30 seconds understandable to a new user.
- Present Vaultarr like a familiar media browser rather than an administration dashboard.
- Keep the animated Blue V Orbital Core as the museum heartbeat.
- Preserve all existing backend features and protected interaction systems.

## Changes
- New museum-lobby Home with Curator story, Continue Browsing, Recently Added, Time Capsule, and Milestones.
- Cleaner Plex-like shelves and restrained dashboard statistics.
- Clear active navigation state.
- Friendlier Discover language and familiar browsing presentation.
- No changes to card tilt, expansion, Focus Mode, manual providers, acquisition, Museum Scan backend, or metadata services.

## Experimental status
This build is intended for comparison and user feedback. The 1.8.4 interface remains the stable reference.
