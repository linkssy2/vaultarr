# Vaultarr 2.0.0 Alpha 57

Alpha 57 rebuilds Settings into a clearer, more attractive Museum Settings Console.

## A calmer settings experience

- A compact overview shows the current experience mode, theme, game-folder count, and access state.
- The Settings Navigator leads directly to Essentials, Game Folders, Appearance, Backup, Access, and advanced Sources.
- Settings are organized into five clearly numbered zones.
- One workspace opens at a time within each zone, reducing visual noise without hiding functionality.
- Active-location feedback makes it clear which settings destination is open.
- Mobile navigation becomes a compact horizontally scrollable settings rail.

## Preserved behavior

- Every existing field, form action, provider option, library control, theme control, and compatibility setting remains available.
- Authentication behavior is unchanged.
- Museum Backup behavior, schedules, restore actions, and archive controls are unchanged.
- Reset safeguards and confirmation requirements are unchanged.
