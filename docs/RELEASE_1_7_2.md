# Vaultarr 1.7.2 — Per-Game Preservation

Preservation is no longer a separate destination users must manage. Each game now carries its own preservation score, badge, missing-item list, and local asset summary.

## Changes

- Added a Preservation section to full game records.
- Renamed the expanded-card Archive Health tab to Preservation.
- Removed Preservation from Advanced sidebar navigation.
- Redirected the legacy `/preservation` route to `/museum?attention=1`.
- Added a Needs Attention Museum filter.
- Changed the Milestones hero action from Preserve More to Improve Your Museum.
- Updated onboarding and documentation to use the new workflow.

The protected 1.5.8 interaction baseline remains unchanged.
