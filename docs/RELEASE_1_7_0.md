# Vaultarr 1.7.0 — Sidebar Museum Scan

Vaultarr now has one obvious place to scan the museum: a permanent **Scan Museum** control directly above **Search Museum** in the sidebar.

When pressed, the button expands in place to display live progress, the current preparation stage, and the active game. When the scan finishes, it briefly displays the completion state and smoothly contracts back into the normal button.

The older Home and Museum scan controls were removed so scanning is no longer duplicated across pages.

The 1.5.8 UX baseline remains protected.
