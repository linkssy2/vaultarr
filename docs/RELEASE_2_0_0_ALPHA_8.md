# Vaultarr 2.0.0 Alpha 8 — Sidebar Utility Alignment

This focused experimental patch aligns the lower sidebar controls without changing their behavior.

## Changes

- Scan Museum now uses the exact same available width as the Current build card.
- Search Museum now uses the exact same available width as the Current build card.
- Removed the extra nested horizontal padding that made both utility pills appear narrower.
- Preserved the existing pill heights, scan animation, search shortcut, sidebar navigation, Museum grid, floating-card motion, and card expansion.
