# Vaultarr 2.0.0 Alpha 43 — Balanced Expanded-Card Navigation

Alpha 43 gives the expanded Museum card a calmer, evenly balanced workspace bar.

## Navigation arrangement

- Overview and Play remain directly accessible.
- Collection contains Information, Manuals, Gallery, Trailer, and Soundtrack.
- Manage contains Preservation, Acquisition, Patches, Files, and Actions.
- The four top-level controls share equal width and spacing at desktop sizes, then settle into a balanced two-column grid on compact screens.

## Preserved behavior

- All existing workspaces and routes remain available.
- Active dropdown state, keyboard navigation, Escape handling, and menu animation are unchanged.
- Expanded-card dimensions, artwork, hover, lift, tilt, gloss, and expansion behavior are unchanged.

## Validation target

- Verify equal top-level control sizing and both dropdown contents in the running Museum.
- Verify Information and Preservation still activate their original workspaces.
- Run Python compilation, template compilation, and JavaScript syntax checks.
