# Vaultarr 1.4.6 — Interaction Reliability Repair

This maintenance release restores consistent pointer, hover, and button behavior across smooth page transitions. Universal Search can be reopened safely while visible, Ctrl/Cmd+K has a single owner, invisible modal layers can no longer intercept the application, and page-specific interactions are reinitialized after every SPA content swap.

The release retains the production GHCR deployment path documented in README.md.
