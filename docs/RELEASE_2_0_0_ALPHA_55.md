# Vaultarr 2.0.0 Alpha 55

Alpha 55 makes Vaultarr accessible and comfortable on mobile devices without redesigning the established desktop application.

## Mobile accessibility

- Compact mobile header with direct access to Museum Scan and universal search.
- Persistent five-destination navigation sized for touch and phone safe areas.
- Responsive two-column museum at common phone widths and a single column on very narrow screens.
- Full-screen expanded exhibits with reachable tabs and bottom-sheet Collection and Manage menus.
- Phone-safe search dialogs, settings shortcuts, dropdowns, tables, forms, collections, milestones, and discovery sections.
- Larger shared touch targets and contained horizontal scrolling for dense controls.
- Clear assistive labels for opening museum cards and closing expanded exhibits.
- Pointer-only museum tilt is suppressed on coarse touchscreens; desktop hover, depth, lift, gloss, and expansion are unchanged.

## Compatibility

- Desktop layouts and protected systems remain unchanged outside responsive breakpoints.
- Theme colors, liquid simulations, routes, sidebar destinations, authentication, and scan behavior are preserved.
