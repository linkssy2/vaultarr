# Vaultarr 2.0.0 Alpha 19 — Adaptive Cover Presentation

This focused release prevents Museum artwork from being cropped to fill fixed card proportions.

## Changes

- Full cover artwork is centered with `object-fit: contain`.
- A blurred, darkened duplicate cover fills any unused card area.
- Portrait, square, landscape, and PC box artwork retain their original proportions.
- No stretching or distortion is introduced.

## Protected systems

No changes were made to floating-card hover, 3D tilt, parallax, gloss, expansion, Focus Mode, Museum Scan, manuals, Acquisition Assistant, or sidebar navigation.
